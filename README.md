# Treinamento Backend: C#

Este repositório é dedicado aos meus estudos e práticas de desenvolvimento de aplicações backend. O foco principal deste projeto é o entendimento do funcionamento interno de APIs, estruturação de banco de dados e lógica de negócios, operando 100% no servidor (sem interface gráfica).

## Habilidades e Conceitos Praticados

Durante o desenvolvimento deste projeto, foquei em aplicar os seguintes conceitos:

* **Domínio do Terminal:** Uso de linha de comando para navegação e execução de scripts.
* **Mapeamento Objeto-Relacional (ORM):** Criação, gerenciamento e execução de **Migrations** para versionamento do banco de dados.
* **Orientação a Objetos:** Modelagem de dados estruturada e criação de classes sólidas.
* **Desenvolvimento de API RESTful:** Construção de um fluxo completo de **CRUD** no backend:
  * `GET`: Consulta e listagem de dados.
  * `POST`: Criação e inserção de novos registros no banco.
  * `PUT`: Atualização de dados existentes.
  * `DELETE`: Remoção segura de registros.

## 🛠️ Tecnologias Utilizadas

* **Linguagens:** C#
* **Bancos de Dados:** MySQL
* **Frameworks:** ASP.NET Core
* **Testes de API:** Swagger
